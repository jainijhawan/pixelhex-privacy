Pixel Hex 소개

Pixel Hex는 디자이너, 개발자 및 크리에이티브 전문가를 위해 설계된 강력한 색상 팔레트 생성기 및 분석기입니다. 이미지에서 아름다운 색상 팔레트를 추출하고, 색상 조화를 탐색하며, 향후 프로젝트를 위해 좋아하는 팔레트를 저장하세요.

📷 이미지 분석

사진을 캡처하거나 선택하여 즉시 주요 색상을 추출하고 아름다운 팔레트를 만드세요.

🎨 색상 선택기

이미지에서 정확하게 색상을 선택하고 자세한 색상 정보를 얻으세요.

💾 저장 및 정리

좋아하는 색상과 팔레트를 쉽게 액세스하고 정리할 수 있도록 저장하세요.

📋 빠른 복사

한 번의 탭으로 여러 형식(HEX, RGB, HSL)의 색상 코드를 복사하세요.

자주 묻는 질문

이미지에서 색상 팔레트를 어떻게 만드나요?

카메라 아이콘 또는 이미지 선택기를 탭하고 사진을 선택하거나 캡처하면 Pixel Hex가 자동으로 이미지를 분석하고 주요 색상에서 아름다운 색상 팔레트를 생성합니다.

색상을 어떻게 저장하나요?

팔레트의 색상을 탭한 다음 저장 아이콘을 탭하세요. 색상이 기기에 로컬로 저장되며 저장된 색상 섹션에서 액세스할 수 있습니다.

색상 코드를 복사할 수 있나요?

네! 색상을 탭하여 세부 정보를 확인한 다음 복사 버튼을 탭하여 색상 코드를 복사하세요. 앱은 HEX, RGB 및 HSL 형식을 지원합니다.

저장된 색상은 어디에 저장되나요?

모든 저장된 색상과 팔레트는 기기에 로컬로 저장됩니다. 서버에 업로드되지 않으므로 개인 정보가 보호되고 오프라인에서도 액세스할 수 있습니다.

앱이 오프라인에서 작동하나요?

네! Pixel Hex는 완전히 오프라인에서 작동합니다. 모든 색상 분석 및 팔레트 생성은 기기에서 로컬로 이루어집니다.

프리미엄 구독에는 무엇이 포함되나요?

프리미엄 기능에는 다음이 포함됩니다:

  • 무제한 저장된 팔레트
  • 고급 색상 분석 도구
  • 색상 조화 생성기
  • 여러 형식으로 팔레트 내보내기
  • 광고 없는 경험
  • 우선 지원

구독을 어떻게 관리하나요?

구독은 Apple ID를 통해 관리됩니다. 구독을 관리하거나 취소하려면:

  1. 기기에서 설정 앱 열기
  2. 상단에서 이름 탭하기
  3. "구독" 탭하기
  4. Pixel Hex 선택
  5. 필요에 따라 관리 또는 취소

시작하기

  1. 앱 실행 - 기기에서 Pixel Hex 열기
  2. 이미지 소스 선택 - 카메라 아이콘을 탭하여 새 사진을 캡처하거나 라이브러리에서 선택
  3. 색상 분석 - 앱이 자동으로 색상을 추출하고 팔레트 생성
  4. 색상 탐색 - 색상을 탭하여 자세한 정보를 확인하고 색상 코드 복사
  5. 즐겨찾기 저장 - 저장 아이콘을 탭하여 나중에 사용할 색상 및 팔레트 저장

문제 해결

앱이 열리지 않거나 시작 시 충돌

  • 최신 버전의 iOS를 실행 중인지 확인
  • App Store에서 Pixel Hex를 최신 버전으로 업데이트
  • 기기 재시작
  • 문제가 지속되면 앱 재설치

카메라 또는 사진 액세스가 작동하지 않음

  • 설정 > 개인정보 보호 및 보안 > 카메라로 이동하여 Pixel Hex가 활성화되어 있는지 확인
  • 설정 > 개인정보 보호 및 보안 > 사진으로 이동하여 Pixel Hex에 액세스 권한이 있는지 확인
  • 권한을 부여한 후 앱 재시작
